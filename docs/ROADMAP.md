# Roadmap

## Milestone 1 — Solar and shadow demo

Complete the independent solar domain, time-zone handling, simplified house,
real-time controls, playback, diagnostics, tests, and GitHub Pages delivery.

## Milestone 2 — Lightweight Pascal JSON parsing

Define a versioned import model, parse supported node metadata and transforms,
report unsupported content, and introduce fixture-based parser tests.

## Milestone 3 — Wall, Slab, Ceiling, Door, Window

Render primary architectural solids and true openings from normalized Pascal
data while retaining material and visibility boundaries.

## Milestone 4 — Roof, Fence, Stair

Add secondary exterior and circulation geometry, bounds-driven shadow cameras,
and large-model performance checks.

## Milestone 5 — Interior navigation and display control

Add interior camera modes, level/discipline visibility, selection diagnostics,
section-style inspection, and model-specific solar study presets.
